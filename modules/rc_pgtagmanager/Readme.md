# Premium Google Tag Manager

## Description

This module allows you to track your site with Google Tag Manager on Prestashop

## Features

- Google Tag Manager
- Google Analytics Enhanced Ecommerce
- Scroll Tracking
- Dynamic Remarketing
- AdWords Conversions
- Bing Conversions
- Facebook Conversions
- Twitter Conversions
- Social network events
